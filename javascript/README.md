
# JavaScript の勉強用ページ

- ここのページを参考にするとゲームができるようにしたい。テキストは後で充実させる
- 簡単なサンプルを置く。まずは動くことが大事。

- 対象としての条件は以下かも。（敷居が高い・・。）
    - HTMLのソースわかる、書ける
    - IF文、FORループとか、関数がわかる
    - コメント「//」とか「/* */」とかわかる
- 基礎的なところはとほほさんのところを見てもらう
    - https://www.tohoho-web.com/html/memo/elem.htm
